# Retinal Vessel Segmentation using U-Net

This project implements a U-Net model for retinal blood vessel segmentation using sample retina images from scikit-image. 
The pipeline includes preprocessing, pseudo-mask generation, U-Net architecture, model training, and prediction visualization.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- OpenCV
- scikit-image
- Matplotlib

## File Structure
- `notebook.ipynb` : Full Colab notebook with training workflow
- `requirements.txt` : Dependencies
- `results/` : Sample outputs (to be uploaded)
